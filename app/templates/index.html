Cafe & Wifi API

Quick Start

Run the server, then test endpoints in your browser (GET only) or using Postman / curl (all methods).

Endpoints

GET
/
Home page (this documentation).
GET
/random
Return one random cafe.
GET
/all
Return all cafes as a JSON list.
GET
/search?loc=<location>
Search cafes by location (query parameter loc).
POST
/add
Add a new cafe (send data as form fields or JSON).
PATCH
/update-price/<cafe_id>?new_price=<price>
Update the coffee price for a cafe by ID.
DELETE
